# Spacecraft Fleet Manager

A React application for viewing spacecraft, building a fleet, and navigating dynamic project routes.

## Features

- View all spacecraft in your fleet
- Build and save new spacecraft
- View spacecraft details using dynamic routes
- Delete spacecraft from the fleet
- Explore planets
- LocalStorage persistence
- React Router navigation
- Responsive CSS design

## Run the project

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.
